How to host on GitHub Pages — free, no code needed
Step 01
Create a GitHub account
Go to github.com and sign up for a free account if you don't have one yet.
Step 02
Create a new repository
Click + → New repository. Name it ere-ad-generator. Set to Public. Click Create repository.
Step 03
Upload this file
Click Add file → Upload files. Drag and drop this file. Rename it to:
index.html
Step 04
Commit the file
Scroll down and click Commit changes. The file is now saved to your repository.
Step 05
Enable GitHub Pages
Go to Settings → Pages. Under Source, select Deploy from a branch → main → / (root). Click Save.
Step 06
Your live URL
After ~1 minute, your generator is live at:
https://[your-username].github.io/ere-ad-generator/
